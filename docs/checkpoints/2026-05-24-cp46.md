**Bucket B — `codegen(b)` `d2fcf3f`** — `Grappa.GenWireTypes` mix task
+ generated wireTypes.ts.

Mix task at `lib/mix/tasks/grappa/gen_wire_types.ex` walks every
module under `lib/grappa/**/wire.ex`, parses `@type` declarations
via `Code.Typespec.fetch_types/1`, emits the deterministic TS file
at `cicchetto/src/lib/wireTypes.ts`. Module ordering alphabetical
by `inspect/1`, types within module in source order (Code.Typespec
returns reversed; we un-reverse).

Type mapping rules: atom literals → TS string literals; atom unions
→ TS string unions; `String.t()`/`Ecto.UUID.t()`/`DateTime.t()` →
`string`; `[T]` → `T[]`; `T | nil` → `T | null`; nested `%{...}` →
`{ ... }`; open map `%{K => V}` → `Record<string, V>`; bare `atom()`
→ `string`; cross-mod ref → `ModSubmodTypeName` alias.

**PLAN DEVIATIONS** (per `feedback_plan_vs_production_reality`):

1. `cicchetto/src/lib/wireTypes.ts` was mounted `:ro` in
   `WORKTREE_VOLUMES`. Added `WRITABLE_CIC=1` escape-hatch in
   `scripts/_lib.sh` matching the existing `WRITABLE_LOCK=1` pattern.
2. TS naming uses fully-qualified `ModSubmodType` form
   (`AdminEventsWireCircuitOpenEvent` not `CircuitOpenEvent`) so
   sibling Wire modules can't collide on bare names like `T` or
   `Event` (visitors/wire + scrollback/wire both declared `t/0`,
   admin_events/wire + scrollback/wire both declared `event/0`).
3. Generated output matches biome `lineWidth: 100`. Long unions
   (`AdminEventsWireEventKind` enum, auto-emitted `Wire<Short>Event`
   payload union) emit multi-line `\n  | a\n  | b;`; short unions
   inline. Top-level + nested unions independently decide via length.
4. Transitive external-type resolution: a `remote_type` reference
   to a non-wire module (e.g. `Networks.Credential.connection_state/0`,
   `Scrollback.Meta.t/0`) gets recorded in a Process-dict registry
   and emitted under `// === External types ===` at the file top.
   Fixpoint-iterates (rendering an external may surface MORE refs);
   depth-limit 8 raises loudly on a pathological cycle.
5. Tightened `render_type/1` + `strip_typespec_metadata/1` specs to
   the inner-AST union Dialyzer infers (Credo wanted @spec, Dialyzer
   wanted narrower than `term()`).

Generated file: 558 lines covering 10 Wire modules + 4 external
types. `bun run check` clean on the generated file (3 pre-existing
BottomBar errors remain, not from this bucket).

ExUnit coverage: 24 tests covering type-mapping per primitive +
fixture-module shapes + discriminated-union emission + deterministic
ordering + `--check` exit codes.

CI green at d2fcf3f.
