### 1. NDP keepalive — more aggressive (`7126c68`)
`infra/freebsd/ndp_keepalive.sh` + rc.d wrapper + `jail_install_rcd.sh`.
- Interval default **30s → 10s**; new `GRAPPA_NDP_KEEPALIVE_COUNT` knob
  (rc var `grappa_ndp_keepalive_count`, default **3**); ping is now a
  `-c ${COUNT} -i 0.5` burst per pool source (was single `-c 1`). Burst
  survives single-packet loss → an entry can't age out on one drop.
- Each pool source now touched ~6×/min vs 2×/min.
- **Live on m42:** rc.d reinstalled, loop +x, `sysrc interval=10 count=3`
  in `/etc/rc.conf.d/grappa_ndp_keepalive`, service restarted (old pid
  18451 → 94697). Verified by catching a live
  `ping -6 -S 2a03:4000:2:33c::442 -c 3 -i 0.5 -W 2000 fe80::1%vtnet0`.
- NOTE: daemon runs with `-f` → loop logs go to /dev/null; verify via
  `ps`-catching a burst, not a logfile. `jail_install_rcd.sh` skips an
  existing rc.conf.d, so live edits needed `sysrc` by hand.

### 2. fail2ban — exempt `/read-cursor` 400s (m42 HOST, not in repo)
The `http-400` jail (`/usr/local/etc/fail2ban/jail.d/defaults.local`,
filter `nginx-bad-request`, maxretry 8) was banning vjt: cic POSTs the
read-cursor with an invalid `message_id` on service-nick query windows
(NickServ ×17 / ChanServ ×10 / OperServ ×4 = 31 × 400). Added a
jail-level `ignoreregex` (keeps the package filter pristine):
```
ignoreregex = ^<HOST> - \S+ \[\] "[^"]*/read-cursor\b[^"]*" 400
```
`\b` so a forged `/read-cursorEVIL` stays bannable. Validated with
`fail2ban-regex` (2 read-cursor lines incl. `?at=N` ignored; EVIL +
TLS-garbage still matched), `reload http-400`, currently-banned 0.
Backup: `defaults.local.bak-readcursor`. **This file lives only on the
host, not in the repo.**

### 3. CSP — allowlist captcha inline bootstrap by hash (`cc5fda7`)
`infra/snippets/security-headers.conf`. The Turnstile/hCaptcha `api.js`
(allowed via its host) injects an INLINE `<script>` to bootstrap the
challenge; strict `script-src` blocked it (`script-src-elem`) → captcha
silently never initialised on prod. Pinned by CSP3 sha256
`'sha256-ZswfTY7H35rbv8WC7NXBoiC7WNu86vSzCDChNWwZZDM='` (NOT
`'unsafe-inline'`). **Live on irc.sniffo.org + irc.sindro.me** (curl'd
script-src confirms the hash). Deployed targeted: pull jail repo →
`install` the one snippet → `nginx -t` → `service nginx reload` (no
BEAM/cic rebuild).
